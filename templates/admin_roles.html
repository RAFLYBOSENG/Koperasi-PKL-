Cara Menambah Role Baru (DB Mode)
  1. Login ke PostgreSQL console
  2. Insert ke tabel roles: 
    INSERT INTO roles (id_role, role_name, deskripsi) 
VALUES (gen_random_uuid(), 'nama_role', 'Deskripsi role');
  3. Mapping permission ke role: 
    INSERT INTO role_permissions (id_role, id_permission) 
SELECT r.id_role, p.id_permission 
FROM roles r, permissions p 
WHERE r.role_name = 'nama_role' 
AND p.permission_name IN ('permission1', 'permission2');
  4. Refresh halaman ini untuk melihat role baru
